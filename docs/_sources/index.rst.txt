Introduction
------------
VitLib is a Python library for image processing and analysis.

Installation
------------
To install VitLib, run the following command:

::

   pip install git+https://github.com/Kotetsu0000/VitLib

Getting Started
---------------
Here is a quick example of how to use VitLib:

.. code-block:: python

   import VitLib
   print(VitLib.USE_CYTHON)
